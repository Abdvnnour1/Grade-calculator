# Grade Calculator

A Python command-line tool that calculates student grade statistics.

## Features
- Input any number of subjects and grades
- Calculates average grade
- Identifies highest and lowest scoring subjects
- Shows pass/fail status per subject (pass mark: 10/20)
- Input validation — rejects grades outside 0–20 range

## How to run
```python functions.py```

- Python 3.14
